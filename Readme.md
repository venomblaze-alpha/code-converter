## 📦 Prerequisites

Ensure you have **[Node.js](https://nodejs.org)** or **[Node.js](https://bun.sh)** installed.

Check your version:

```bash
node -v
# or
bun -v
```
If not installed, grab it from nodejs.org or bun.sh.

🛠️ Installation
1. Clone the Repository
```bash
git clone https://github.com/venomblaze-alpha/code-converter.git
cd code-converter
```

2. Install Dependencies
Using npm:
```bash
npm install
```
Or with Bun:
```bash
bun install
```

3. Start the Dev Server
Using npm:
```bash
npm run dev
```
Or with Bun:
```bash
bun run dev
```
Open your browser and visit:
```bash
http://localhost:3000
```
